# Dataset Description  
* `X16k.mat`,  16k data, 5 inputs `[ bitsPerSym, JtoS, EbNo, IntfType,duty]`  
* `Y16k.mat`,  16k data, 2 inputs `[ unMitBer, MitBer]`   
* `X160k.mat`, 160k data input  
* `Y160k.mat`, 160k data output  
Notice: data has beed shuffled.
